# Tasia Viewer

**This is a custom viewer build for personal use. It is not intended or supported for anyone else.**

- Built from the [Firestorm](https://www.firestormviewer.org/) source
- **Second Life grids are removed** — this viewer cannot log into Second Life
- Configured for OpenSim grids only
- No proprietary components (FMOD, KDU, BugSplat, code signing)
- No official releases, no support, no documentation

If you are not the intended user, this repository will not be useful to you.
