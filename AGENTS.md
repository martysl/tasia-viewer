# Project Agent Rules

Before working, read these files if present:

1. `PROJECT_STATUS.md`
2. `NEXT_ACTION.md`
3. `BUILD_NOTES.md`
4. `DECISIONS.md`
5. `SECRETS_POLICY.md`

Rules:

- Use the repo files as source of truth.
- Do one bug/change at a time.
- Do not expose secrets.
- Ask before destructive git actions, sudo, or authenticated operations.
- Update `PROJECT_STATUS.md` and `NEXT_ACTION.md` after meaningful progress.
